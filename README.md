# ร้านยาออนไลน์ของคุณ — Static Site

เว็บ static สำหรับแสดงแคตตาล็อกยาสามัญประจำบ้านและผลิตภัณฑ์สุขภาพ พร้อมเพจเกี่ยวกับ/ติดต่อ/นโยบาย เหมาะสำหรับใช้งานกับ GitHub Pages

## โครงสร้างไฟล์
- `index.html` — หน้าแรก มีช่องค้นหาและรายการสินค้า
- `assets/styles.css` — สไตล์หลักของเว็บไซต์
- `assets/script.js` — สคริปต์เรนเดอร์สินค้าและค้นหา
- `pages/about.html` — เกี่ยวกับเรา
- `pages/contact.html` — ช่องทางติดต่อ/สั่งซื้อ (แก้ไข LINE/อีเมล/โทร)
- `pages/policy.html` — นโยบาย/ข้อกำกับ

## การปรับแต่งอย่างรวดเร็ว
1) เปลี่ยนชื่อร้าน/ไตเติล
- แก้ใน `index.html` ส่วน `<title>` และลิงก์ที่มีข้อความ “ร้านยาออนไลน์ของคุณ”
- แก้ในเพจย่อยทุกไฟล์ใต้ `pages/`

2) แก้ไขสินค้า
- เปิด `assets/script.js` แล้วแก้ไขอาเรย์ `products` (id, name, desc, price, unit, tags)
- เพิ่ม/ลบรายการได้อิสระ

3) ช่องทางติดต่อ
- เปิด `pages/contact.html` แล้วใส่ LINE ID / อีเมล / เบอร์โทรของคุณ

## Deploy ขึ้น GitHub Pages
1. สร้างรีโพบน GitHub และ push โค้ดนี้ขึ้นสาขา `main`
2. ไปที่ Settings → Pages
3. ใน “Build and deployment” เลือก Source = “Deploy from a branch”
4. เลือก Branch = `main` และ Folder = `/ (root)` → กด Save
5. รอให้ระบบ build เสร็จ จากนั้นหน้าเว็บจะอยู่ที่ `https://<username>.github.io/<repo>/`

เคล็ดลับ: ถ้าต้องการโดเมนเอง ให้เพิ่ม DNS และไฟล์ `CNAME` ที่ root (ใส่โดเมนของคุณหนึ่งบรรทัด)

## ข้อกำกับ/ความปลอดภัย
- เว็บไซต์นี้แสดงเฉพาะ “ยาสามัญประจำบ้านและผลิตภัณฑ์สุขภาพทั่วไป”
- ไม่จำหน่ายยาควบคุมพิเศษ/ยาที่ต้องใช้ใบสั่งแพทย์ทางเว็บไซต์
- ผู้ใช้ควรอ่านฉลากก่อนใช้ยา และปรึกษาเภสัชกร/แพทย์เมื่อจำเป็น
- โปรดแสดงข้อมูลผู้รับอนุญาต/ที่ตั้ง/ช่องทางติดต่อจริงหากต้องตามกฎหมายที่เกี่ยวข้อง
